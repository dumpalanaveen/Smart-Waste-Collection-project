Frequently Asked Questions

How do I get started?

Navigate to the Dashboard page and start by viewing existing bins. You can then run route optimization using either algorithm.

Which algorithm should I use?

A* Search is faster and works well for smaller networks (up to 50 bins). Genetic Algorithm is better for larger networks and complex scenarios.

How do I update bin fill levels?

You can update bins manually from the Dashboard or use the Dataset & Simulation page to run automated simulations.

Can I compare algorithm performance?

Yes! Visit the Models & Comparison page to run side-by-side comparisons and view detailed performance metrics.
